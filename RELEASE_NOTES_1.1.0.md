# 家庭电影院 1.1.0

发布日期：2026-06-07

## 新增功能

- 新增 JavBus 刮削方式，补充 DMM / DMM2 覆盖不到的影片信息。
- 新增 Javzimu、AVSubtitles、迅雷字幕等外挂字幕搜索来源，并将迅雷设为默认字幕搜索方式。
- 新增播放器外挂字幕样式设置，可调整字号、底部位置和背景透明度。
- 网盘浏览新增按大小排序，可在时间排序和大小排序之间切换。

## 优化

- 切换外挂字幕时不再从头重新播放，字幕会更自然地加载。
- JavBus 图片下载自动携带 Referer，避免图片 403。
- 网盘添加入库流程更稳定，图片下载失败不会提前删除 STRM，避免影片库出现无法删除的异常记录。
- 文件夹也参与网盘大小排序，查找大文件更方便。
- 115 网盘设置移除 MissAV Cookie 手动控件，相关信息改为需要时自动处理。
- 演员页面移除手动更新头像控件，页面更简洁。
- 优化影片库整理、刷新和重复记录处理逻辑。

## 修复

- 修复 JavBus 番号搜索时错误补零的问题，例如 `DLDSS-00289` 会按 `DLDSS-289` 搜索。
- 修复图片下载失败时可能导致旧 STRM 被删除、详情页无法删除影片的问题。
